# thethaomammo

Tournament management app — Next.js 16 + Supabase + Upstash, free-tier focused.

> **⚠️ Educational rewrite — not for production use.**
> This codebase is a learning exercise / greenfield rewrite study. It has not been
> security-audited for real-world deployment, has known launch-gated items (see
> `docs/development-roadmap.md`), and ships without the operational hardening a
> production tournament platform would require. Use at your own risk; no warranty.
